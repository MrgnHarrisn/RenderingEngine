# BoxEngine

A modern, lightweight **C++ game engine** built from scratch with a focus on learning, performance, and clean architecture.  
Uses **Premake** for building.

---

## Features

- Written in **modern C++ (C++17)**

> *TO BE DONE*
- Modular engine architecture
- Rendering system OpenGL
- Input handling
- Asset loading (textures, models, shaders)
- Editor support (optional / WIP)
- Cross-platform build setup using **Premake**

> **Status:** Actively developed / Hobby project / Experimental

##  Build Instructions

Install [Premake](https://premake.github.io/download) and add it to your path

```
premake5 vs2022 # (window)
premake5 gmake  # (linux)
```
